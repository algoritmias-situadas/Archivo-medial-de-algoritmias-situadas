# **Polyphonic Theme II**



Polyphonic Theme II es una pieza programada en SuperCollider, realizada para la clase de Composición Asistida por Computadora. La obra explora procesos polifónicos a partir de rutinas y estructuras generadas por código.



------



//// Ejecutar la pieza



Primero debes abrir el archivo "1. Polyphonic Theme 2.scd" en un IDE de SuperCollider.



Al abrir el archivo, debes de comenzar ejecutando el primer bloque de código (entre paréntesis). Este bloque abre y compila otros archivos ".scd" necesarios para que la pieza funcione correctamente.



Después, ejecuta el segundo bloque de código (también entre paréntesis), el cual corresponde a la rutina principal de la pieza y pone en marcha el desarrollo musical.



Es importante ejecutar los bloques en este orden.
