# **Retrato sonoro de guitarristas en búsqueda de atención**



Retrato sonoro de guitarristas en búsqueda de atención es una pieza programada en SuperCollider. La obra funciona como una exploración sonora y performativa a partir de una interfaz gráfica y un sistema de entradas y salidas multicanal.



------



//// Requisitos previos



Para poder ejecutar la pieza es necesario instalar plugins adicionales para SuperCollider, ya que se utiliza el UGen "coyote.kr".



Las instrucciones completas de instalación se encuentran en el siguiente enlace:



\[https://github.com/supercollider/sc3-plugins](https://github.com/supercollider/sc3-plugins)



------



//// Ejecutar la pieza



Abre el archivo "Amplificador (Click Aqui).scd" en un IDE de SuperCollider.



1\. Ejecuta el primer bloque de código (entre paréntesis).

&nbsp;  Este bloque configura el sistema en octofonía, habilitando 8 salidas y 8 entradas, aunque para la pieza solo se utilizan 3 entradas.



2\. Después, ejecuta el segundo bloque de código (también entre paréntesis).

&nbsp;  Este bloque abre el GUI, desde donde se puede acceder a la interfaz de la pieza y comenzar la interacción sonora.



Es importante ejecutar los bloques en este orden.
